About the website

  • This is a site designed to help users prepare a shopping list.

  • Each product contains 3 features: Category, Name and Amount.

  • The products are sorted by ABC by category name and then by product name. This can help save unnecessary walking to the same shopping area.

  • The site is easy and convenient to operate. You can add, edit and delete a product. All data is stored in the database and will be loaded every time a user logs on to the site.

  • You can also search for a product by name or by category. The system will reload all products that match the user's search request.
